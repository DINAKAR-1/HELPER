2. Functional Interfaces
Interfaces with exactly one abstract method.
Predicate<String> startsWithD = s -> s.startsWith("D");
System.out.println(startsWithD.test("Dinakar")); // true

3. Streams API
Operations:
Intermediate: filter, map, sorted
Terminal: collect, forEach, reduce
